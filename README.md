# Cotton

A React Native boilerplate project built with Expo, TypeScript, NativeWind, Jotai, and Back4App Parse Server.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. Open in Expo Go on your device or simulator

## Tech Stack

- **React Native** with Expo
- **TypeScript** for type safety
- **NativeWind v4** for Tailwind CSS styling
- **Jotai** for state management
- **Back4App Parse Server** for backend
- **Expo Router** for navigation

## Project Structure

- `src/app/` - Expo Router screens
- `src/components/` - Reusable UI components
- `src/hooks/` - Custom React hooks
- `src/services/` - API services
- `src/store/` - Jotai atoms
- `src/types/` - TypeScript types
- `cloud/` - Back4App Cloud Code

See `CLAUDE.md` for detailed documentation.
